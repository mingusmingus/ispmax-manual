# Registrar pago

<Lead>Permite ingresar un pago de forma manual. Se busca al cliente, se selecciona la factura correspondiente y se elige el tipo de pago.</Lead>

<Figure src="/assets/imgs/registrar-pago.webp" alt="Registrar pago" caption="Formulario para registrar un pago manualmente." width={1645} height={370} />

<Steps>
<Step>

Busca al cliente por razón social o número de documento.

</Step>
<Step>

Selecciona la factura que se desea pagar.

</Step>
<Step>

Elige el tipo de pago (registrar y activar u otra opción disponible).

</Step>
<Step>

Verifica el valor y guarda el registro.

</Step>
</Steps>

:::danger Precaución

Antes de registrar un pago, verifica en **Pagos registrados** o **Transacciones** que no exista un registro previo para la misma factura. Evita duplicados.

:::
