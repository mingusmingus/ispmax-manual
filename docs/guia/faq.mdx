# Preguntas frecuentes

:::details ¿Por qué un cliente aparece sin deuda?

Puede no tener facturas pendientes o sus facturas ya están pagadas. Revisa la pestaña **Facturación / Pagos** dentro del perfil del cliente para confirmar el estado real.

:::

:::details ¿Qué hago si un pago no aparece en el listado?

Busca primero en **Pagos registrados** y en **Transacciones** usando filtros de fecha, operador y forma de pago. No registres otro pago hasta confirmar que no existe uno previo.

:::

:::details ¿Por qué un router aparece sin conexión?

Puede deberse a un problema de red, credenciales API incorrectas o puerto bloqueado. Usa **Comprobar conexión** en el router y, si el error persiste, comunícalo al administrador técnico.

:::

:::details ¿Qué significa el estado Pendiente en una factura?

Indica que hay un pago parcial registrado o que la factura está en un estado intermedio. Revisa el saldo detallado antes de tomar cualquier acción sobre esa factura.

:::

:::details ¿Qué módulos debo usar con precaución?

Ajustes en general, especialmente: **Configuración de Cortes**, **Cambios Masivos**, **Roles y Permisos**, **Mensajería masiva**, **Formas de Pago** y **Eliminar datos**. Úsalos solo con autorización del administrador.

:::

:::details ¿Cómo busco un cliente si no recuerdo su nombre exacto?

Prueba buscando por número de documento, IP asignada, número de teléfono o parte del apellido. Los filtros del listado de clientes permiten combinar criterios.

:::
