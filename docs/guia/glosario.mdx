# Glosario rápido

<Cards columns={3}>
<Card title="Cliente">

Persona o empresa que recibe el servicio de internet del ISP.

</Card>
<Card title="Servicio">

Conexión o plan de internet asignado a un cliente específico.

</Card>
<Card title="Router">

Equipo MikroTik que administra y controla las conexiones de los clientes.

</Card>
<Card title="Plan">

Paquete de internet con velocidad de descarga/subida y precio mensual definidos.

</Card>
<Card title="IP">

Dirección de red asignada al cliente o equipo para identificarlo en la red.

</Card>
<Card title="CIDR">

Notación de máscara de subred (ej. /24 equivale a 254 IPs disponibles).

</Card>
<Card title="Factura">

Documento generado por el sistema con el valor a cobrar por el servicio.

</Card>
<Card title="Pago">

Registro del dinero recibido por un servicio o concepto facturado.

</Card>
<Card title="Deuda">

Valor acumulado pendiente de cancelar por parte del cliente.

</Card>
<Card title="Ticket">

Solicitud o reporte de soporte técnico vinculado a un cliente.

</Card>
<Card title="NAP">

Caja de distribución de fibra óptica instalada en campo.

</Card>
<Card title="VLAN">

Segmentación lógica de red para separar tráfico por tipo o usuario.

</Card>
<Card title="OLT">

Equipo central de la red de fibra óptica que gestiona las ONUs.

</Card>
<Card title="ONU">

Equipo en el domicilio del cliente que recibe la señal de fibra óptica.

</Card>
<Card title="API">

Interfaz de conexión que permite a ISPMAX comunicarse con routers y plataformas externas.

</Card>
<Card title="SMTP">

Protocolo usado para enviar correos electrónicos desde el sistema.

</Card>
<Card title="Burst">

Velocidad de ráfaga temporal que supera el límite normal del plan por un tiempo definido.

</Card>
<Card title="FE">

Factura Electrónica. Documento tributario autorizado por el SRI.

</Card>
</Cards>
