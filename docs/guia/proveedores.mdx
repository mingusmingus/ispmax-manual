# Proveedores

<Lead>Permite registrar y administrar los proveedores del ISP: empresas o personas a quienes se realizan compras o pagos. Los proveedores registrados aquí pueden asociarse en Compras e Ingresos/Egresos.</Lead>

| Campo | Descripción |
| --- | --- |
| **Razón Social** | Nombre comercial o legal del proveedor. |
| **Tipo de Documento** | RUC, Cédula, Pasaporte u otro. |
| **Número de Documento** | Identificación fiscal del proveedor. |
| **Email** | Correo de contacto. |
| **Celular** | Número de contacto. |
| **Dirección** | Dirección física del proveedor. |
| **Provincia / Ciudad** | Ubicación del proveedor. |
