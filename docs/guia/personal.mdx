# Gestión de personal

<Lead>Permite administrar los usuarios del sistema ISPMAX: crear, editar, activar o desactivar cuentas, y configurar sus accesos y permisos.</Lead>

<Figure src="/assets/imgs/personal-listado.webp" alt="Gestión de personal" caption="Listado de usuarios: nombre, rol, email, estado y acciones." width={1645} height={301} />

<Figure src="/assets/imgs/personal-edicion.webp" alt="Editar usuario" caption="Configuración de usuario: datos, rol, permisos, routers, sucursal y horario de acceso." width={1669} height={752} />

| Campo | Descripción |
| --- | --- |
| **Nombre / Apellidos** | Identificación del usuario. |
| **Usuario** | Nombre de usuario para iniciar sesión. |
| **Email** | Correo electrónico del usuario. |
| **Teléfono móvil** | Número de contacto del operador. |
| **Rol** | Nivel de acceso: Técnico, Administrador o Super Administrador. |
| **Permisos Router** | Routers que el usuario puede ver y gestionar. |
| **Sucursal** | Sucursal a la que pertenece o tiene acceso el usuario. |
| **Estado** | Activo o Inactivo. |
| **Foto** | Imagen de perfil del usuario. |
| **Horario de acceso** | Días y rango horario en que el usuario puede iniciar sesión. |
| **Contraseña** | Campo para asignar o cambiar la contraseña del usuario. |
| **Permisos por módulo** | Control granular sobre qué puede ver o hacer en Dashboard, Gestión de Red, Gestión de Clientes, Finanzas y Ajustes. |
