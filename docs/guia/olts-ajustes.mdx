# OLTs

<Lead>Permite configurar funciones generales para SmartOLT y AdminOLT, como el reinicio automático de ONUs y la validación de equipos registrados en cada plataforma.</Lead>

<Figure src="/assets/imgs/olts-ajustes.webp" alt="Ajustes OLTs" caption="Configuración de reinicio automático y validación de ONUs para SmartOLT y AdminOLT." width={1591} height={680} max="1591px" />

:::tip Consejo

**Reinicio automático de ONUs** — Programa reinicios periódicos con intervalo configurable (ej. cada 5 días) y hora de ejecución. Se configura de forma independiente para SmartOLT y AdminOLT.

:::

:::tip Consejo

**Validación de ONUs** — Cruza los servicios registrados en ISPMAX contra los vinculados en la plataforma OLT. Marca como inválidos los que ya no existen. Útil para detectar desincronizaciones.

:::
