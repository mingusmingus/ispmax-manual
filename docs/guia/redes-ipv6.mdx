# Redes IPv6

<Lead>Permite registrar rangos de direcciones IPv6 para los servicios que utilizan este protocolo. Funciona de forma análoga a Redes IPv4.</Lead>

<Figure src="/assets/imgs/ipv6-listado.webp" alt="Redes IPv6" caption="Listado de redes IPv6 con columnas y estados." width={1649} height={186} max="1649px" />

<Figure src="/assets/imgs/ipv6-formulario.webp" alt="Nueva red IPv6" caption="Formulario para registrar una nueva red IPv6." width={1558} height={279} max="1558px" />

| Columna / Campo | Descripción |
| --- | --- |
| **Nombre** | Nombre identificador de la red. |
| **Red** | Dirección base en formato IPv6 (ej. `2803:2540:1c::`). |
| **Máscara** | Prefijo de la red (ej. 50). |
| **Subred** | Tamaño de subred asignado (ej. 64). |
| **Router** | Equipo al que pertenece esta red. |
| **Estado** | <Badge type="tip">Activo</Badge> o <Badge type="danger">Inactivo</Badge>. |

:::tip Consejo

El lápiz edita el registro. El ícono de prohibido inhabilita la red para que no sea usada en nuevas asignaciones.

:::
