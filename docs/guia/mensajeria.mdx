# Mensajería

<Lead>Permite configurar WhatsApp y correo electrónico para enviar mensajes individuales o masivos a clientes. Tiene dos secciones: **Configuración** (credenciales de los canales) y **Mensajes Masivos** (redacción y envío).</Lead>

<Figure src="/assets/imgs/mensajeria-config.webp" alt="Configuración mensajería" caption="Configuración de WhatsApp y datos del servidor de correo (SMTP)." width={1631} height={466} max="1631px" />

:::tip Consejo

**Vincular WhatsApp** — Conecta un número de WhatsApp al sistema para habilitarlo como canal de envío.

:::

:::tip Consejo

**Servidor de correo** — Configura host, puerto, usuario, contraseña y tipo de seguridad (SSL/STARTTLS/sin cifrado). Incluye botón para probar la configuración antes de guardar.

:::

<Figure src="/assets/imgs/mensajeria-masivos.webp" alt="Mensajes masivos" caption="Redactor de mensajes masivos con variables, canal de envío e intervalo." width={1628} height={663} max="1628px" />

:::tip Consejo

**Filtros** — Permite segmentar el envío por router, red IPv4, plan y estado del cliente.

:::

:::tip Consejo

**Redactor** — Editor de texto con formato para escribir el mensaje. Soporta variables de datos dinámicos del cliente: nombre, cédula, celular, email, plan, valor, día de pago, deuda, estado y empresa.

:::

:::tip Consejo

**Canal de envío** — WhatsApp, Correo electrónico o Ambos.

:::

:::tip Consejo

**Intervalo entre mensajes** — Tiempo mínimo entre cada envío (mínimo 30 segundos) para evitar bloqueos. El historial de mensajes enviados queda registrado para consulta posterior.

:::

:::warning Atención

Lee el aviso de uso responsable antes de enviar mensajes masivos. Un envío mal configurado puede alcanzar a clientes incorrectos o saturar el canal.

:::
