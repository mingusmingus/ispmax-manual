# Sistema

<Lead>Permite ejecutar la actualización de ISPMAX desde el servidor. Al presionar **Actualizar Sistema**, se inicia el proceso de actualización de la plataforma a la versión más reciente disponible.</Lead>

:::danger Precaución

**Importante:** ejecuta la actualización únicamente cuando el equipo lo indique o cuando sea necesario. No actualices durante horarios críticos de facturación o atención. Asegúrate de tener respaldo y autorización del administrador antes de proceder.

:::
