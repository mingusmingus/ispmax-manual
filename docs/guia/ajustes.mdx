# Panel de ajustes

<Lead>Reúne las opciones de configuración sensibles del sistema. Deben ser utilizadas únicamente por personal autorizado, ya que pueden afectar facturación, cortes, permisos, mensajería y datos maestros.</Lead>

<Figure src="/assets/imgs/ajustes-panel.webp" alt="Panel de ajustes" caption="Acceso a todos los módulos de configuración avanzada del sistema." width={1540} height={402} />

:::danger Precaución

**Solo usuarios avanzados:** no modifiques opciones de Ajustes sin conocer su efecto. Un cambio incorrecto puede afectar cortes automáticos, facturas, permisos de usuarios o el funcionamiento general del sistema.

:::
