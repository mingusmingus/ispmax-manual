# Ingresos / Egresos

<Lead>Registra movimientos de dinero que no corresponden directamente al cobro mensual de internet: gastos operativos, ingresos adicionales, pagos a proveedores o compromisos periódicos. También permite crear movimientos recurrentes para gastos o ingresos que se repiten.</Lead>

<Figure src="/assets/imgs/ingresos-panel.webp" alt="Panel ingresos egresos" caption="Resumen financiero: total ingresos, total egresos del día y periodo." width={1670} height={847} max="1670px" />

<Figure src="/assets/imgs/ingresos-formulario.webp" alt="Formulario ingreso egreso" caption="Formulario para registrar un nuevo ingreso o egreso." width={496} height={702} max="496px" variant="compact start" />

| Campo | Descripción |
| --- | --- |
| **Tipo** | **Ingreso** (dinero que entra) o **Egreso** (dinero que sale). |
| **Router** | Router o zona relacionada con el movimiento, si aplica. |
| **Fecha** | Fecha en que ocurrió el movimiento. |
| **Monto** | Valor del ingreso o gasto. |
| **Proveedor** | Proveedor relacionado, si corresponde. |
| **Recurrente** | Indica si el movimiento se repite: una sola vez, mensual, anual, etc. |
| **Referencia** | Código, número de comprobante u otra referencia del movimiento. |
| **Archivo adjunto** | Soporte documental del gasto o ingreso (facturas, recibos, etc.). |
| **Detalles** | Nota explicativa adicional. |
