# Servicios personalizados

<Lead>Permite crear servicios o cargos adicionales que no corresponden a un plan de internet estándar. Se usa para registrar conceptos especiales de facturación adaptados a necesidades particulares: descuentos fijos, cargos por instalación o servicios complementarios.</Lead>

<Figure src="/assets/imgs/personalizado-listado.webp" alt="Servicios personalizados" caption="Listado de perfiles personalizados activos." width={1656} height={276} />

<Figure src="/assets/imgs/personalizado-formulario.webp" alt="Formulario perfil personalizado" caption="Formulario para crear un servicio o cargo personalizado." width={490} height={477} max="370px" variant="compact start" />

| Campo | Descripción |
| --- | --- |
| **Nombre** | Nombre del concepto (ej. DESCUENTO POR DISCAPACIDAD). |
| **Precio** | Valor numérico. Puede ser monto fijo o porcentaje según configuración. |
| **Impuesto** | Tipo de impuesto aplicable (ej. Sin IVA). |
| **ICE** | Impuesto a Consumos Especiales si aplica. |
| **Descripción** | Texto explicativo del concepto (opcional). |

:::tip Consejo

El lápiz edita el perfil. El ícono de basurero lo elimina permanentemente del sistema.

:::
