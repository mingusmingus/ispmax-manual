# SmartOLT

<Lead>Integra ISPMAX con la plataforma externa SmartOLT para la administración de redes de fibra óptica. Permite consultar ONUs, VLANs, zonas, perfiles de velocidad y ejecutar acciones sobre los equipos de la red óptica.</Lead>

:::warning Atención

Para que esta sección funcione, la integración debe estar configurada con el dominio y API Key de SmartOLT en la pestaña **API** de esta sección. Sin esta configuración, los datos no se cargarán.

:::

| Sección / Pestaña | Descripción |
| --- | --- |
| **ONUs Configurados** | Listado de ONUs vinculadas desde SmartOLT. Muestra estado, nombre, SN, OLT, board, puerto, señal e IPv4. Permite autorizar, habilitar, deshabilitar, reiniciar y ver información técnica de cada ONU. |
| **VLANs** | VLANs configuradas organizadas por OLT. Al agregar una VLAN se especifica ID, descripción y uso: IPTV, MGMT/VOIP, DHCP Snooping o LAN to LAN. |
| **Zonas** | Agrupan ONUs por área geográfica o segmento de red. El nombre de la zona debe coincidir exactamente con el nombre registrado en SmartOLT. |
| **Profiles** | Perfiles de velocidad disponibles en SmartOLT, divididos en Perfiles de Descarga y Perfiles de Subida. Se sincronizan desde SmartOLT. |
| **API** | Configuración de la integración. Requiere: **Subdominio** (ej. miempresa o miempresa.smartolt.com) y **API Key**. |
