# Detalle de cliente

<Lead>Al abrir un cliente se despliega una ficha con pestañas que contienen toda la información del servicio, facturación, soporte y actividad.</Lead>

<Figure src="/assets/imgs/clientes-detalle.webp" alt="Detalle de cliente" caption="Detalle de cliente" width={1571} height={640} max="1571px" />

| Pestaña | Contenido |
| --- | --- |
| **Resumen** | Datos personales y generales del cliente. |
| **Servicios** | Planes activos, IP asignada, router, contrato y acciones técnicas disponibles. |
| **Tickets** | Solicitudes o problemas reportados y su estado actual. |
| **Documentos** | Contratos firmados y archivos adjuntos del cliente. |
| **Facturación / Pagos** | Historial de facturas, deuda, pagos y comprobantes. |
| **Logs** | Registro de acciones realizadas sobre el cliente. |

## Acciones comunes

| Acción | Para qué sirve |
| --- | --- |
| **Ver servicios** | Revisar plan, IP, router, estado del servicio y datos técnicos. |
| **Ver facturación** | Consultar facturas, pagos, deuda y comprobantes del cliente. |
| **Crear ticket** | Registrar una solicitud de soporte o problema reportado por el cliente. |
| **Ver documentos** | Revisar contratos, archivos adjuntos o documentos relacionados. |
| **Revisar logs** | Consultar cambios o acciones realizadas sobre el cliente. |
