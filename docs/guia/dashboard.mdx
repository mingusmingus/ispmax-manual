# Dashboard

<Lead>Es la pantalla inicial. Muestra un resumen del estado financiero y operativo del ISP: clientes, servicios, cobros, facturas pendientes y últimos movimientos.</Lead>

<Figure src="/assets/imgs/vista-general-del-dashboard.webp" alt="Dashboard ISPMAX" caption="Dashboard ISPMAX" width={1679} height={777} variant="hero" />

| Bloque | Qué muestra | Cómo interpretarlo |
| --- | --- | --- |
| **Total de clientes / servicios** | Cantidad de clientes registrados y servicios creados. | Un cliente puede tener más de un servicio activo, por eso ambos valores pueden diferir. |
| **Facturas pagadas** | Monto cobrado y número de pagos del periodo. | Refleja la recaudación del mes o periodo seleccionado. |
| **Transacciones hoy** | Cobros registrados durante el día. | Ideal para seguimiento de caja diaria. |
| **Facturas no pagadas** | Monto pendiente y cantidad de facturas sin cancelar según el periodo o filtro aplicado. | Úsalo como alerta de cartera vencida. |
| **Resumen de facturación** | Comparación entre mes actual y mes pasado. | Compara mes actual y mes pasado. Puede mostrar facturas pagadas, pendientes y no pagadas según los filtros seleccionados. |
| **Últimos pagos** | Pagos registrados recientemente. | Permite confirmar operaciones recientes sin entrar al módulo de transacciones. |

:::tip Consejo

Usa los filtros de **Router** y **Sucursal** para ver indicadores de una zona específica. Con `TODOS` se muestra información consolidada.

:::
