# Roles, empresa y sucursales

<Lead>Agrupa tres configuraciones relacionadas con la estructura del ISP en el sistema.</Lead>

<Figure src="/assets/imgs/roles-empresa.webp" alt="Roles y empresa" caption="Listado de roles del sistema y formulario de datos de la empresa." width={1616} height={335} max="1616px" />

<Figure src="/assets/imgs/sucursales.webp" alt="Sucursales" caption="Listado de sucursales con datos de establecimiento y punto de emisión." width={1568} height={387} max="1568px" />

<Cards columns={3}>
<Card title="Roles y permisos">

Define los niveles de acceso del sistema. Los tres roles por defecto son: **Técnico**, **Administrador** y **Super Administrador**. Se pueden crear roles personalizados con el botón **Agregar Rol**.

</Card>
<Card title="Empresa">

Contiene los datos del ISP: RUC, nombre comercial, dirección, email, celular, logo y si requiere facturación electrónica. Esta información aparece en los comprobantes generados.

</Card>
<Card title="Sucursales">

Puntos de atención o emisión del ISP. Cada sucursal tiene dirección, número de establecimiento, punto de emisión y número de factura. Se agregan desde el botón **Agregar Sucursal**.

</Card>
</Cards>

:::warning Atención

Modificar roles, datos de empresa o sucursales puede cambiar permisos de usuarios, información visible en comprobantes o la secuencia de numeración de facturas.

:::
