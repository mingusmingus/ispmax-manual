# Primeros pasos

<Lead>Para ingresar a ISPMAX necesitas un usuario y contraseña proporcionados por el administrador. Cada usuario accede únicamente a los módulos que su rol permite.</Lead>

<Cards columns={3}>
<Card title="1. Ingresa al sistema">

Abre la dirección de ISPMAX en tu navegador, escribe tu usuario y contraseña, y presiona **Iniciar sesión**.

</Card>
<Card title="2. Revisa tu rol">

En la parte superior verás tu rol. Este define qué módulos puedes ver y qué acciones puedes realizar.

</Card>
<Card title="3. Usa los filtros">

La mayoría de módulos permite filtrar por router, sucursal, fecha o estado. Úsalos antes de exportar reportes o tomar decisiones.

</Card>
</Cards>

:::warning Atención

No compartas tus credenciales. Si otra persona necesita acceso, el administrador debe crear un usuario con permisos propios desde **Gestión de Personal**.

:::
