# Promesas de pago

<Lead>Registra el compromiso de un cliente para cancelar una factura o deuda en una fecha acordada. Permite dar seguimiento sin perder el historial del acuerdo.</Lead>

<Figure src="/assets/imgs/promesas-pago.webp" alt="Promesas de pago" caption="Listado de promesas activas y formulario para registrar una nueva promesa." width={1638} height={739} max="1638px" />

- **Buscar cliente:** selecciona el cliente que realiza el compromiso.
- **Factura a pagar:** elige la factura pendiente relacionada.
- **Fecha límite de pago:** día acordado para cancelar la deuda.
- **Nota:** resumen breve del acuerdo registrado.
