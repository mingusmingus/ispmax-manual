# Transacciones

<Lead>Concentra los movimientos de pago registrados. Es la vista principal para verificar cobros, imprimir comprobantes, revisar el historial financiero y generar reportes de cierre.</Lead>

<Figure src="/assets/imgs/transacciones-listado.webp" alt="Transacciones" caption="Listado de transacciones con totales: subtotal, IVA, total cobrado y total neto." width={1647} height={733} />

Los filtros disponibles permiten buscar por forma de pago, detalle de forma de pago, operador, fecha, horario, sucursal, estado de pago, tipo de comprobante, router y red. Después de filtrar, puedes descargar el reporte en PDF o Excel o revisar el detalle de cada transacción.

:::tip Consejo

Las acciones disponibles por transacción varían según el rol: imprimir comprobante, editar, anular o ver detalle.

:::
