# Listado de clientes

<Lead>Es la tabla principal de consulta de clientes. Permite buscar y filtrar por nombre, IP, plan, dirección, documento, deuda, estado de pago y estado de servicio.</Lead>

<Figure src="/assets/imgs/clientes-listado.webp" alt="Listado de clientes" caption="Listado de clientes" width={1646} height={863} max="1646px" />

| Columna | Qué muestra |
| --- | --- |
| **Cliente** | Nombre o razón social del cliente. |
| **IPv4** | Dirección IP asignada al servicio. |
| **Plan** | Plan de internet contratado. |
| **Dirección servicio** | Lugar donde está instalado el servicio. |
| **Último pago** | Fecha y hora del último pago registrado. |
| **Deuda actual** | Valor pendiente acumulado del cliente. |
| **Estado pago** | <Badge type="tip">Pagado</Badge> · <Badge type="warning">Pendiente</Badge> · <Badge type="danger">No pagado</Badge> |
| **Estado servicio** | <Badge type="tip">Activo</Badge> · <Badge type="warning">Suspendido</Badge> · <Badge type="danger">Retirado</Badge> |

:::tip Consejo

Si no encuentras al cliente por nombre, busca por documento, IP, teléfono o apellido.

:::

## Cómo buscar un cliente

<Lead>Puedes buscar un cliente usando diferentes datos disponibles en el sistema.</Lead>

- Nombre o apellido.
- Número de documento.
- Dirección IP asignada.
- Número de teléfono.
- Plan contratado.
- Estado de pago o estado de servicio.

:::tip Consejo

Si no encuentras al cliente por nombre completo, intenta buscar solo por apellido, IP, documento o teléfono.

:::
