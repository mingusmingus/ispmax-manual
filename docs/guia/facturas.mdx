# Facturas

<Lead>Permite consultar las facturas emitidas por el sistema, revisar su estado y generar nuevas cuando sea necesario. Es uno de los módulos más utilizados para auditoría y control financiero.</Lead>

<Figure src="/assets/imgs/facturas-listado.webp" alt="Listado de facturas" caption="Listado de facturas: filtros por fecha, tipo, sucursal y estado." width={1671} height={270} />

<Figure src="/assets/imgs/facturas-agregar.webp" alt="Agregar factura" caption="Formulario para generar una nueva factura." width={1659} height={749} />

| Campo | Uso |
| --- | --- |
| **Fecha de emisión** | Día en que se genera la factura. |
| **N° Factura** | Identificador secuencial del documento. |
| **Cliente** | Persona o empresa a quien se factura. Por defecto: CONSUMIDOR FINAL. |
| **Sucursal** | Punto de emisión del comprobante. |
| **Forma de pago** | Canal de cobro seleccionado. |
| **Productos / servicios** | Ítems que componen la factura. Se buscan por nombre o código de barras. |
| **Subtotal / IVA / ICE / Descuento** | Desglose de impuestos y descuentos aplicados. |
| **Total de venta** | Valor final a cobrar. |
| **Plazo de pago** | Días o fecha límite para cancelar (opcional). |
| **Enviar mensaje al cliente** | Activa el envío automático del comprobante al cliente. |
