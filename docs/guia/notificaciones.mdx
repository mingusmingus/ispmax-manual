# Tipos de Notificaciones

En esta sección aprenderá a configurar las notificaciones de recordatorio de pago en ISPMAX. El sistema ofrece dos métodos de configuración: de forma general para toda la cartera de clientes o de manera individual para clientes específicos. Como buena práctica, se recomienda realizar envíos masivos segmentados en grupos de hasta 60 clientes por día para mejorar la estabilidad del servicio y la efectividad de las notificaciones.

## Métodos de Configuración

El sistema ofrece dos métodos de configuración: de forma general para toda la cartera de clientes o de manera

**Configuración General:** permite establecer recordatorios automáticos para todos los clientes del sistema.

**Configuración por Cliente:** permite personalizar los recordatorios de pago para clientes específicos según sus necesidades.
