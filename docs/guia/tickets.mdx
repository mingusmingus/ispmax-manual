# Tickets de soporte

<Lead>Los tickets permiten registrar solicitudes, fallas o trabajos técnicos. Dan seguimiento desde que se reporta el problema hasta que se resuelve, y quedan como historial de atención al cliente.</Lead>

<Figure src="/assets/imgs/tickets-listado.webp" alt="Tickets de soporte" caption="Tickets de soporte" width={1655} height={198} max="1655px" />

<Cards columns={3}>
<Card title="Crear ticket">

Desde el perfil del cliente o desde el módulo Tickets. Se registra asunto, prioridad, técnico asignado y descripción del problema.

</Card>
<Card title="Seguimiento">

El estado se actualiza según el avance: **Abierto → En proceso → Resuelto → Cerrado**.

</Card>
<Card title="Historial">

Sirve como evidencia de la atención y soporte brindado al cliente a lo largo del tiempo.

</Card>
</Cards>
