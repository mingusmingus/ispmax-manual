# Cajas NAP

<Lead>Las cajas NAP representan los puntos físicos de distribución de fibra óptica instalados en campo. Permiten organizar puertos, registrar detalles técnicos de instalación y localizar los equipos desde el mapa.</Lead>

<Figure src="/assets/imgs/nap-listado.webp" alt="Cajas NAP listado" caption="Listado de cajas NAP con columnas y estados." width={1654} height={222} />

<Figure src="/assets/imgs/nap-formulario.webp" alt="Formulario Caja NAP" caption="Formulario para registrar una nueva caja NAP." width={1586} height={513} />

| Campo | Descripción |
| --- | --- |
| **Nombre** | Nombre identificador de la caja. |
| **Ubicación** | Referencia física: calle, sector o poste. |
| **Coordenadas** | Longitud y latitud para ubicar la caja en el mapa. |
| **Puertos** | Número de puertos disponibles en la caja. |
| **Detalles** | Referencia adicional (ej. Poste 5). |
| **Estado** | <Badge type="tip">Activo</Badge> · <Badge type="warning">En mantenimiento</Badge> · <Badge type="info" outline>Desactivada</Badge> |
| **Potencia** | Nivel de señal óptica registrado en la caja. |
| **Detalles adicionales** | Información complementaria: color de hilo, número de splitter. |
| **Router** | Router al que pertenece esta caja NAP. |
