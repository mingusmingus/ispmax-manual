# Base de datos

<Lead>Agrupa herramientas avanzadas para adaptar y gestionar la información del sistema: campos personalizados, cambios masivos y administración de ubicaciones.</Lead>

<Figure src="/assets/imgs/base-datos-campos.webp" alt="Campos personalizados" caption="Crear campos personalizados y herramienta de eliminación masiva de datos." width={1570} height={643} />

<Figure src="/assets/imgs/base-datos-cambios.webp" alt="Cambios masivos" caption="Cambios masivos: actualización de configuración para múltiples clientes a la vez." width={1562} height={630} />

| Sección | Qué hace |
| --- | --- |
| **Campos personalizados** | Agrega campos extra al perfil de los clientes para registrar información específica del ISP. El nombre del campo usa guiones bajos en lugar de espacios (ej. `numero_contrato`). Una vez creado, aparece en la ficha de todos los clientes. |
| **Cambios masivos** | Actualiza configuraciones de facturación para muchos clientes simultáneamente: día de pago, tipo, impuesto, días de gracia, tipo comprobante y más. Solo modifica los campos editados; los que se dejan en blanco no cambian. |
| **Ubicaciones** | Gestiona provincias, cantones y parroquias disponibles en los formularios del sistema. |
| **Eliminar datos** | Permite borrar registros de una tabla seleccionada, filtrados por router, red o plan. Acción irreversible; requiere respaldo previo. |

:::danger Precaución

**Atención:** los cambios masivos y la eliminación de datos son irreversibles. Asegúrate de tener un respaldo antes de ejecutar cualquier acción en esta sección.

:::
