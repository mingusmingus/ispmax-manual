# Personas

<Lead>Vista general de personas registradas en el sistema. Permite buscar, editar y agregar registros con datos básicos de identificación y contacto.</Lead>

<Figure src="/assets/imgs/personas-listado.webp" alt="Listado de personas" caption="Listado de personas: nombre, tipo de documento, email, móvil y estado." width={1457} height={610} max="1457px" />

<Figure src="/assets/imgs/personas-formulario.webp" alt="Agregar persona" caption="Formulario para registrar una nueva persona." width={713} height={563} max="400px" variant="compact start" />

| Campo | Descripción |
| --- | --- |
| **Cliente** | Nombre completo de la persona. |
| **Tipo de Documento** | Cédula, RUC, Pasaporte u otro. |
| **Número de Documento** | Número de identificación. |
| **Celular** | Número de contacto con código de país. |
| **Email** | Correo electrónico de la persona. |
| **Dirección** | Dirección física. |
| **Provincia / Cantón / Parroquia** | Ubicación geográfica del registro. |
