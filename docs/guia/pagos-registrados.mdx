# Pagos registrados

<Lead>Muestra el historial completo de pagos. Es el módulo principal para cierres de caja, auditorías y revisión de cobros por operador, fecha o forma de pago.</Lead>

<Figure src="/assets/imgs/pagos-filtro-forma.webp" alt="Filtro forma de pago" caption="Filtros por forma de pago y detalle de forma de pago." width={1649} height={805} max="1649px" />

<Figure src="/assets/imgs/pagos-filtro-operador.webp" alt="Filtros pagos registrados" caption="Operador, rango de fechas, horario, sucursal y estado de pago." width={1614} height={342} max="1614px" />

<Figure src="/assets/imgs/pagos-comprobante-reporte.webp" alt="Comprobante y reporte" caption="Tipo comprobante (Recibo/Factura), estado de factura electrónica y descarga de reporte." width={205} height={144} max="205px" variant="compact start" />

| Filtro | Uso |
| --- | --- |
| **Forma de pago** | Filtra por efectivo, bancos, instalaciones u otra forma configurada. |
| **Detalle forma de pago** | Precisa el canal: efectivo oficina/sucursal, transferencia, depósito, etc. |
| **Operador** | Persona que registró el pago en el sistema. |
| **Fechas y horario** | Rango de búsqueda por día y franja horaria. |
| **Sucursal** | Punto de atención donde se procesó el pago. |
| **Estado pago** | No anulados, anulados o generados automáticamente. |
| **Tipo comprobante** | Recibo o Factura. |
| **Estado FE** | Autorizados o No autorizados por el SRI. |
| **Reporte** | Descarga PDF o Excel del resultado filtrado. |
