# Uso de Plantillas

Las plantillas permiten importar información masiva a ISPMAX mediante archivos Excel prediseñados.

## Orden de Carga de Plantillas

1. Redes-ipv4.
2. redes-ipv6.
3. Planes y servicios.
4. Clientes.
