# Productos y servicios

<Lead>Permite administrar los productos físicos y servicios adicionales del ISP: nombre, código de barras, stock disponible, descuento, precio de compra, precio de venta (PVP) y estado.</Lead>

<Lead>Los productos pueden ser materiales físicos, equipos, accesorios o servicios adicionales que luego pueden usarse en compras o facturación.</Lead>

<Figure src="/assets/imgs/productos-listado.webp" alt="Productos y servicios" caption="Listado de productos con código, stock, precios y estado." width={1621} height={294} max="1621px" />

| Campo | Descripción |
| --- | --- |
| **Nombre** | Nombre del producto o servicio. |
| **Código de barras** | Código usado para identificar o buscar el producto rápidamente. |
| **Stock** | Cantidad disponible en inventario. |
| **Precio de compra** | Valor al que se adquirió el producto. |
| **PVP** | Precio de venta al público. |
| **Descuento** | Descuento configurado si aplica. |
| **Aplica IVA** | Indica si el producto o servicio genera IVA. |
| **Estado** | Indica si el producto está activo o inactivo. |

:::tip Consejo

Útil para materiales de instalación, equipos, accesorios y servicios extras que pueden incluirse en compras o en facturación directa al cliente.

:::
