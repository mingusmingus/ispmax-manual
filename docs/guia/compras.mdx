# Compras

<Lead>Permite registrar los productos o materiales adquiridos por la empresa. Ayuda a controlar gastos, llevar historial de proveedores e impactar el inventario cuando corresponde. Ejemplos de uso: compra de rollos de fibra, conectores, equipos o accesorios de instalación.</Lead>

<Figure src="/assets/imgs/compras-listado.webp" alt="Listado de compras" caption="Listado de compras con filtros por fecha, sucursal y estado." width={1658} height={284} />

:::tip Consejo

Al registrar una compra se puede indicar el proveedor, número de comprobante, descripción, productos, cantidades, descuentos, IVA y total. Usa los filtros por fecha y sucursal para consultar compras anteriores.

:::

| Campo | Descripción |
| --- | --- |
| **Proveedor** | Persona o empresa a quien se realiza la compra. |
| **Sucursal** | Punto donde se registra la compra. |
| **Número de comprobante** | Referencia del documento recibido. |
| **Productos** | Artículos, materiales o equipos comprados. |
| **Cantidad** | Número de unidades adquiridas. |
| **Descuento / IVA** | Valores aplicados a la compra según corresponda. |
| **Total** | Valor final de la compra registrada. |
