# Formas de pago

<Lead>Permite crear y organizar los canales de cobro disponibles en el sistema. Cada forma de pago puede agrupar varios detalles específicos.</Lead>

<Figure src="/assets/imgs/formas-pago-listado.webp" alt="Formas de pago" caption="Listado de formas de pago con nombre, detalle y estado." width={1597} height={554} max="1597px" />

:::tip Consejo

**Forma de pago** — Grupo principal que categoriza el canal. Ejemplo: Pago de Mensualidad Efectivos, Pago de Mensualidad Bancos, Instalaciones.

:::

<Figure src="/assets/imgs/formas-pago-detalle.webp" alt="Detalle formas de pago" caption="Canales específicos asociados a cada forma de pago." width={1597} height={728} max="1597px" />

:::tip Consejo

**Detalle de forma de pago** — Canal específico dentro de la forma. Ejemplo: Efectivo Oficina/Sucursal, Transferencia Bancaria, PagoFácil, Depósito.

:::
